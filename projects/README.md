# projects/

Local clones of upstream terminals, kept **unmodified**. These are
gitignored (see `../.gitignore`) — clone them yourself:

```sh
git clone https://github.com/ghostty-org/ghostty      projects/ghostty
git clone https://github.com/wezterm/wezterm          projects/wezterm
git clone https://github.com/tmux/tmux                projects/tmux
git clone https://github.com/kovidgoyal/kitty         projects/kitty
git clone https://github.com/microsoft/vscode         projects/vscode
```

Any research change goes in a separate branch or an `experiments/*/patch.diff`,
never committed back into this repo.
